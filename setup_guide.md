## 1. スプレッドシートの作成

> **作業者：夫のGoogleアカウントでログインした状態で行う**

### 手順

1. ブラウザで [Google スプレッドシート](https://sheets.google.com) を開く
2. 左上の「**＋ 新しいスプレッドシート**」をクリック
3. スプレッドシートのタイトルを変更する（例：`サロン家計簿データ`）
4. URL をコピーして保存しておく（後で使います）
   ```
   例: https://docs.google.com/spreadsheets/d/【ここがスプレッドシートID】/edit
   ```
   `/d/` と `/edit` の間の長い文字列が **スプレッドシート ID** です

> シートの中身（「取引」「設定」シートの作成）は、GASのセットアップ関数が自動で行います

## 2. GASコードの貼り付け・デプロイ

> **作業者：夫のGoogleアカウント**

### 2-1. Apps Script を開く

1. 作成したスプレッドシートを開いた状態で
2. メニューの「**拡張機能**」→「**Apps Script**」をクリック
3. Apps Script のエディタが別タブで開く

### 2-2. コードを貼り付ける

1. エディタ左側の「コード.gs」をクリック
2. **既存の内容をすべて削除**する（Ctrl+A → Delete）
3. `gas.gs` ファイルの内容を全部コピーして貼り付ける

### 2-3. スプレッドシート ID を設定する

貼り付けたコードの上部にある以下の行を編集する：

```javascript
// 変更前
const SPREADSHEET_ID = 'YOUR_SPREADSHEET_ID_HERE';

// 変更後（手順1でコピーしたIDに書き換える）
const SPREADSHEET_ID = '1BxiMVs0XRA5nFMdKvBdBZjgmUUqptlbs74OgVE2upms';
```

4. 右上の「💾 保存」ボタン（またはCtrl+S）でコードを保存

### 2-4. セットアップ関数を実行する（初回のみ）

1. エディタ上部の関数選択ドロップダウンで「**setupSheets**」を選ぶ
2. 「▶ 実行」ボタンをクリック
3. **「承認が必要です」と表示されたら：**
   - 「権限を確認」→ 夫のGoogleアカウントを選択
   - 「詳細」→「サロン家計簿データ（安全ではないページ）に移動」をクリック
   - 「許可」をクリック
4. 「セットアップ完了！」のダイアログが表示されれば成功
5. スプレッドシートに「取引」「設定」シートが自動作成されたことを確認

### 2-5. ウェブアプリとしてデプロイする

1. エディタ右上の「**デプロイ**」→「**新しいデプロイ**」をクリック
2. 歯車アイコン⚙️をクリックして「**ウェブアプリ**」を選択
3. 以下のように設定する：

   | 項目 | 設定値 |
   |------|--------|
   | 説明 | サロン家計簿 API v1（任意） |
   | 次のユーザーとして実行 | **自分**（夫のアカウント名） |
   | アクセスできるユーザー | **全員** |

   > ⚠️「全員」にしないと妻がログインなしで使えません

4. 「**デプロイ**」をクリック
5. 「ウェブアプリ URL」が表示される → **コピーして保存する**
   ```
   例: https://script.google.com/macros/s/AKfycbxxx.../exec
   ```
6. 「完了」をクリック

## 3. 妻アカウントへの共有設定

> **作業者：夫のGoogleアカウント**
> ※ データをスプレッドシートで直接確認したい場合の任意設定です

1. スプレッドシートを開く
2. 右上の「**共有**」ボタンをクリック
3. 妻のGmailアドレスを入力
4. 権限を「**編集者**」に設定
5. 「送信」をクリック

> アプリ（index.html）の操作は共有設定なしでも使えます。
> スプレッドシートの中身を妻が直接見たい場合のみ必要です。

## 4. アプリへのURL設定

> **作業者：妻（または夫）。スマホ・PCどちらでも同じ手順**

1. `index.html` をブラウザで開く
2. 下部ナビの「**設定**」タブをタップ
3. 画面上部に「🔗 Google スプレッドシート連携」セクションが表示される
4. 「**GAS Web アプリ URL**」欄に、手順2-5でコピーした URL を貼り付ける
5. 「**URL を保存**」ボタンをタップ → 「✅ URL を保存しました」と表示される
6. 「**接続テスト**」ボタンをタップ
   - 「✅ 接続成功！」と表示されれば完了
   - 「❌ 接続に失敗」の場合は下記トラブルシューティング参照

### 動作確認

1. 「記録する」タブで収支を1件入力して保存
2. スプレッドシートの「取引」シートに行が追記されることを確認
3. 「収支一覧」で削除 → スプレッドシートから行が消えることを確認

## 5. 保守・再デプロイ時の手順

> **作業者：夫のGoogleアカウントのみで完結**

### gas.gs のコードを変更した場合

> ⚠️「新しいデプロイ」を選ぶと URL が変わってしまいます。
> 必ず「**デプロイを管理**」から既存デプロイを更新してください。

1. Apps Script エディタでコードを修正・保存
2. 「**デプロイ**」→「**デプロイを管理**」をクリック
3. 一覧に表示された既存のデプロイの右の「鉛筆✏️アイコン」をクリック
4. 「バージョン」を「**新しいバージョン**」に変更
5. 「**デプロイ**」をクリック
6. URL は変わらないため、アプリ側の設定変更は不要

### app.js / style.css を変更した場合

1. ファイルを修正
2. `index.html` / `style.css` / `app.js` の3ファイルをまとめて同じフォルダに置き直す
3. ブラウザで `index.html` を再度開けばすぐ反映される（URLの再設定は不要）

### スプレッドシートのバックアップ

- スプレッドシートは Google ドライブに自動保存されています
- 念のためアプリの「設定」→「データをバックアップ（JSON）」でローカルにも保存できます

## トラブルシューティング

### 「❌ 接続に失敗しました」が表示される

| 確認箇所 | 対処 |
|---------|------|
| GAS のデプロイ設定「アクセスできるユーザー」 | 「全員」になっているか確認 |
| SPREADSHEET_ID の設定 | gas.gs の `SPREADSHEET_ID` が正しいか確認 |
| URL のコピーミス | デプロイ管理画面から URL を再コピー |
| GAS の実行権限 | `setupSheets` 関数を一度実行して権限を付与 |

### オフライン時の動作

- スマホが圏外・Wi-Fiなし → 端末にデータを一時保存
- 画面上部に「未同期 ○件」バッジが表示される
- オンラインに復帰すると自動でスプレッドシートへ同期される

### 複数端末で使う場合

- スマホとPC両方で使う場合：それぞれ同じ URL を設定画面から設定する
- どちらのデバイスも同じスプレッドシートに書き込まれるため、データは共有される
