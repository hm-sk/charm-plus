## 1. コンセプト・ゴール

### ターゲット
個人経営または小規模の美容サロン（ネイル・エステ・まつエク・ヘアサロン等）

### 解決する課題
| 現状の課題 | Charm+ で解決 |
|-----------|-------------|
| 予約はLINEや電話で口頭管理 | 顧客がWebフォームから24時間予約可能 |
| 顧客情報がノート・頭の中 | デジタルカルテで施術履歴一元管理 |
| 売上・経費はExcelや手書き | 自動集計・青色申告連携 |
| ツールがバラバラで二重入力 | 予約→来店→売上まで自動連携 |

### デザインコンセプト
**"Luxe Minimal"** ― 高級感と使いやすさを両立

| 要素 | 現在 | 新デザイン |
|-----|------|----------|
| カラー | ピンク系 | シャンパンゴールド + ディープチャコール（`#1C1C1E`）|
| アクセント | 濃いピンク | ゴールド（`#B8966E`）、セージグリーン（`#8A9E85`）|
| フォント | system-ui | 見出し: Cormorant Garamond / 本文: Noto Sans JP |
| 印象 | かわいい | エレガント・プロフェッショナル |

## 3. モジュール設計

### 3-1. 予約管理モジュール（新規）

#### 顧客向け予約フォーム（`booking.html`）
- **公開URL**：GASでホストまたは静的HTMLとして配布
- ログイン不要で誰でもアクセス可能
- 入力項目：
  - 名前・電話番号・メール
  - 希望メニュー（マスタから選択）
  - 希望日時（カレンダーUI、埋まっている枠は選択不可）
  - 備考（アレルギー等）

#### 管理画面側（予約タブ）
- 週次/月次カレンダービュー
- 予約ステータス管理：`pending` → `confirmed` → `completed` / `cancelled`
- 予約確認メール自動送信（GmailAPI経由）
- 当日リマインダー通知

#### データモデル（Sheetsシート：`予約`）
```
id | createdAt | customerName | phone | email | menu | price
dateTime | duration | status | notes | reminderSent
```

---

### 3-2. 顧客管理モジュール（新規）

#### 機能一覧
- 顧客一覧・検索（名前・電話番号）
- デジタルカルテ（施術内容・使用薬剤・写真）
- 来店履歴（予約→売上と自動連携）
- 次回来店提案日（最終来店 + メニューの推奨間隔）
- 誕生日・記念日メモ（DM用途）
- カルテ写真：Google Drive アップロード（既存 `uploadReceipt` と同じ仕組み）

#### データモデル（Sheetsシート：`顧客`）
```
id | createdAt | name | nameKana | phone | email | birthday
notes | allergyNotes | lastVisit | visitCount | totalSpend
drivePhotoFolderId
```

#### データモデル（Sheetsシート：`カルテ`）
```
id | customerId | appointmentId | date | menu | detail
productUsed | photoFileIds | staffNote
```

---

### 3-3. 売上・経費管理モジュール（既存 → 拡張）

#### 既存機能（v2.0.0）
- 収支記録、テンプレート、家事按分、自動仕訳ルール
- GAS連携、領収書アップロード

#### 拡張予定
- **予約完了 → 売上自動入力**：予約をcompletedにすると売上レコードを自動生成
- **顧客ごとの売上分析**：リピーター率、客単価推移
- **月次ダッシュボード**：売上グラフ、経費比率、粗利
- **青色申告帳票出力**：現金出納帳CSVエクスポート

## 5. 技術的な制約と判断

| 項目 | 方針 |
|-----|------|
| 認証 | オーナー管理画面はパスワード保護（GAS側でシンプルな認証トークン）|
| 予約フォーム | 認証不要・公開URL。スパム対策にreCAPTCHAも検討 |
| データ上限 | Sheetsは100万行が上限。数百人規模のサロンには十分 |
| オフライン | 管理画面はLocalStorage一時保存→オンライン復帰時に同期（既存） |
| メール送信 | GasのMailApp（無料、1日100通）。大量送信時はSendGridに切り替え |
| LINEBot連携 | Phase 4以降で検討。LINE Messaging API + GASで実現可能 |
